Le Programme
Questions
❦
Concrètement, qu'est-ce que StoryVoice fait ? +
Vous chargez un ebook. StoryVoice l'analyse : il détecte les personnages, les lieux et leurs relations, rédige des résumés, et peut narrer le texte avec une voix différente par personnage. Le tout dans une seule application web (installable sur PC et mobile).
Mes ebooks personnels sont-ils privés ? +
Oui. Vos fichiers restent dans votre navigateur et ne sont jamais envoyés sur nos serveurs. Seules les voix sont synthétisées à distance, à partir du texte lu.
C'est gratuit ou payant ? +
La lecture augmentée — résumés, carte des personnages, questions sans spoiler — est gratuite. Seule la narration vocale premium se règle, via des crédits de voix (sans abonnement).
J'ai déjà acheté un code — où je le colle ? +
Ouvrez l'application, puis ⚙ Réglages → Code d'accès et collez le code reçu par e-mail (il commence par sv_). Un souci ? Écrivez à quangapps.dev@gmail.com.
