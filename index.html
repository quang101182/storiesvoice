Chapitre VI
Le Programme
❦
Mes ebooks personnels sont-ils privés ? +
Oui, entièrement. Vos fichiers restent dans votre navigateur et ne sont jamais envoyés sur nos serveurs. Seules les voix sont synthétisées à distance, à partir du texte que vous lisez — le fichier, lui, ne bouge pas.
Puis-je changer les voix des personnages ? +
Oui. StoryVoice propose une distribution par défaut, et vous pouvez réattribuer les voix à votre goût pour chaque personnage.
Que se passe-t-il quand mes crédits sont épuisés ? +
La lecture augmentée (résumés, carte, questions) continue de fonctionner gratuitement. Seule la narration audio se met en pause : il vous suffit de reprendre un billet quand vous le souhaitez. Aucun prélèvement automatique.
J'ai déjà acheté un code — où je le colle ? +
Ouvrez l'application, puis ⚙ Réglages → Code d'accès et collez le code reçu par e-mail (il commence par sv_). Votre solde de voix apparaît aussitôt. Un souci ? Écrivez à quangapps.dev@gmail.com.
